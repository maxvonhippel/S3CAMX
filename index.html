Falsification of Safety Properties for Closed Loop Control Systems

This project is aimed at analyzing both the controller code and the plant dynamical system in order to search for violating trajectories to given properties.

Simulate and Scatter + Symbolic Execution

'Simulate and Scatter' is used to generate and examine plant behaviors using a model which can be simulated, while symbolic execution is used to maximize the path coverage in the controller code.
